
# Master Coffee Pro - Reqres & AI Integration

Assignment project for Frontend Development using React, Reqres API, and Google Gemini AI.

## Daftar Library yang Digunakan
- **React 19**: Library utama untuk membangun antarmuka pengguna.
- **Vite**: Build tool dan development server yang cepat.
- **Tailwind CSS (via CDN)**: Framework CSS untuk desain responsif dan styling premium.
- **@google/genai**: SDK resmi untuk integrasi dengan Gemini 3 (Coffee Sommelier).
- **Font Awesome**: Library ikon untuk navigasi dan elemen UI.
- **Reqres API**: Digunakan sebagai backend untuk autentikasi dan data user.

## Fitur Utama (Requirements)
- **Register & Login**: Autentikasi menggunakan endpoint Reqres (e.g., `eve.holt@reqres.in`).
- **List Users**: Menampilkan daftar "Global Baristas" dari API Reqres.
- **Pagination**: Implementasi navigasi antar halaman pada daftar user.
- **User Detail**: Modal detail untuk melihat profil lengkap barista (ID, Email, Avatar).
- **Protected Routes**: Halaman "Staff Portal" hanya muncul dan bisa diakses jika user sudah login.
- **Responsive Design**: Tampilan yang optimal di perangkat mobile maupun desktop.

## Fitur Tambahan (Di luar Requirement)
- **AI Coffee Sommelier**: Chatbot bertenaga Gemini 3 yang membantu memilih jenis kopi.
- **Premium Aesthetic**: Desain dengan tema "Coffee Shop" menggunakan palet warna Stone & Amber.
- **Session Persistence**: Status login tersimpan di `sessionStorage` sehingga tidak hilang saat refresh.
- **Interactive Stats**: Section pencapaian coffee shop dengan animasi sederhana.

## Cara Menjalankan Proyek
1. Pastikan Node.js sudah terinstal.
2. Jalankan `npm install` untuk menginstal dependensi.
3. Jalankan `npm run dev` untuk memulai server development.
4. Buka browser di `http://localhost:5173`.
